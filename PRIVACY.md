# Privacy Policy

`tweetTweet` 是一個用於展示 UI 與資料流的作品集專案,並非提供給一般使用者的商業服務,也不在 App Store 上架。

## 不收集任何資料

這個 App:

- 不收集任何個人資料
- 不收集使用紀錄、設備資訊、位置資訊
- 不使用任何分析工具(Analytics)
- 不使用任何廣告或行為追蹤 SDK
- 不會將資料上傳到任何伺服器

## 資料來源

App 內顯示的貼文、頭像、圖片都來自專案內附的本地 JSON 與圖檔,不會從網路抓取。

## 本地儲存

使用者在 App 中的互動(發文、留言、按讚、追蹤等)只會在當下的 App 記憶體中更新,**不會寫入任何持久化儲存**(沒有 UserDefaults、CoreData、檔案系統等寫入)。關掉 App 後重啟即恢復為原始展示資料。

## 第三方服務

App 沒有整合任何第三方服務、登入、雲端或社群平台。

## 聯絡

如有疑問,請透過此 repository 的 issue 提出。
